# Support

Thanks for using Career-Ops Capture.

## Questions and help

For questions, setup help, and general discussion, use
[GitHub Discussions](https://github.com/rubicon/career-ops-capture/discussions).
Please do not open issues for questions. The issue tracker is reserved for
confirmed bugs and feature requests so it stays actionable.

## Bugs and feature requests

If you have found a reproducible bug or want to request a feature, open an issue
using the templates at
[github.com/rubicon/career-ops-capture/issues](https://github.com/rubicon/career-ops-capture/issues).
Include your browser and version, the extension version, and clear steps to
reproduce.

## Security

Do not report security vulnerabilities through public issues or discussions. See
[SECURITY.md](SECURITY.md) for the private reporting process.
